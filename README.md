## 🛠️ Built With

- **HTML5:** Semantic architecture to ensure a clean structure.
- **CSS3:** Advanced styling featuring smooth neon-glow animations, glassmorphism, responsive grids, and interactive hover states.
- **Vanilla JavaScript:** Event-driven logic, DOM manipulation, custom cursor logic, loading screens, and an interactive background canvas.
- **Dynamic Assets:** Integrated with Devicon & Boxicons for crisp utility and technology icons.

## ✨ Key Features

- **Interactive Animations:** Beautiful entry staggers, a sleek loading state, neon-glowing typography, and a custom mouse cursor.
- **Responsive Layout:** Elegantly scales from desktop monitors down to mobile devices.
- **Skills Categorization:** Visually dynamic skills breakdown spanning Languages, Frontend, Backend, Database/Cloud, and DevOps.
- **Project Showcase:** Detailed cards for my best applications including *AI Teacher*, *UniMart*, and *AI Review Summarizer*—complete with live links.
- **Modern UI/UX:** Includes an auto-typing text effect, scroll indicators, image lightboxes for certificates & CVs, and a polished contact form with floating labels.
